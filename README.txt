推奨環境
OS : windows
Ver: python3.12
仮想環境: python venv

---- 使う前に ----

仮想環境を作成する:
bash
python3.12 -m venv env

仮想環境をアクティブにする:
Windows:
C:\Users\**> env\Scripts\activate

macOS/Linux:
source env/bin/activate

必要な依存パッケージをインストール:
python3.12 -m pip install -r requirements.txt
